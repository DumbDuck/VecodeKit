在 Android 项目中使用导出的代码

如何导出 Android graphics code，请参考这里。

假设我们导出 Android graphics code 时选择文件名为 PictureTiger，包名为 com.example 就导出一个文件 PictureTiger.java，其中定义类 com.example.PictureTiger。

package com.example.PictureTiger;

import android.graphics.*;
import com.vecodekit.GraphicsPicture;

public class PictureTiger extends GraphicsPicture
{
    xxxxx
}

此时我们还缺少了 com.vecodekit.GraphicsPicture 的代码，这代码可以在 VecodeKit 中找到。

获取 VeocdeKit

您可以在 GitHub 下载到最新的 VecodeKit 代码，也可以点击菜单项 Copy VecodeKit... 将其复制到本机中。

解压后，在 VecodeKit/Android/src 下可以找到 com.vecodekit 的代码。另外 VecodeKit/Android/VecodeKitExample 是一个 Android Studio 的工程例子，演示了怎么使用导出的绘图代码。

使用 GraphicsPicture

将 VecodeKit/Android/src 下的代码添加到您的项目中，并且将导出的 Android graphics code 文件也添加到项目中。

比如导出了 PictureTiger.java 文件，定义了 com.example.PictureTiger 类。PictureTiger.java 添加到项目后，就可以调用函数：

PictureTiger tiger = new PictureTiger();
tiger.drawInRect(rect, canvas, PictureContentMode.ScaleAspectFit);

将图片绘制到 canvas 上。

或者使用如下代码，将 tiger 转成 Bitmap。

Bitmap bitmap = tiger.transToBitmap(100, 100, PictureContentMode.ScaleAspectFit);

GraphicsPicture API

getWidth，getHeight

public final float getWidth();
public final float getHeight();

这两个函数取得图片原始大小。比如您需要绘制 2 倍大小 Bitmap，可以使用如下代码：

float bitmapWidth = tiger.getWidth() * 2;
float bitmapHeight = tiger.getHeight() * 2;
Bitmap bitmap = tiger.transToBitmap(bitmapWidth, bitmapHeight);

drawInRect

这个是最核心的函数，将 picture 绘制到 canvas 上面。

public final void drawInRect(RectF rect, Canvas canvas, int contentMode);

rect 表示图片绘制的矩形大小；canvas 是绘图环境。最后一参数 contentMode 表示了如何缩放图片以便适应传入的 rect，可以有如下数值。

public class PictureContentMode
{
    public static final int ScaleToFill       = 0;
    public static final int ScaleAspectFit    = 1;
    public static final int AspectFill        = 2;
    public static final int Center            = 3;
}

transToBitmap

public final Bitmap transToBitmap(float width, float height, int contentMode);

这个函数可以将 GraphicsPicture 转换成 Bitmap。最后一参数 contentMode 的含义见上面。
